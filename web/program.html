Цель курса

Разобраться в основных понятиях,связанных с языком разметки HTML и каскадными таблицами стилей CSS. Получить навыки создания web-страниц, с использованием HTML и CSS.

Вы поймете значение основных тегов HTML,сможете без усилий стилизировать и размещать элементы с помощью CSS и сверстаете свой первый макет
